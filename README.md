## 院长镜像加速器
1. 将需要拉取的镜像写入images.txt
2. Actions将自动触发拉取并推送至阿里云镜像仓库


### images.txt 格式要求每行格式支持两种：
#### 直接写镜像名：nacos/nacos-server:latest
#### 带平台参数：--platform linux/amd64 nacos/nacos-server:latest
